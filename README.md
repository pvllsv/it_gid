# Проект "IT-GID"

Это одностраничный информационный сайт, созданный для помощи родителям в выборе IT-курсов для своих детей. Сайт предоставляет обзоры, сравнения и руководства по различным направлениям в IT-обучении.

## Технологический стек

*   **HTML5** - для структуры и разметки.
*   **SASS** - для написания стилей с последующей компиляцией в CSS, с использованием методологии БЭМ для именования классов.
*   **JavaScript (ES6+)** - для интерактивности и логики на стороне клиента.
*   **jQuery** - как основная JavaScript-библиотека для упрощения DOM-манипуляций.
*   **Slick Carousel** - плагин для создания адаптивных слайдеров.
*   **Anitabs** - плагин для реализации динамических вкладок.

## Структура проекта

- `it_gid/index.html`: Главный и единственный HTML-файл.
- `it_gid/css/`: Папка для скомпилированных CSS-файлов и стилей библиотек.
  - `style-min.css`: Основной файл стилей, сгенерированный из SASS.
- `it_gid/sass/`: Исходные SASS-файлы.
  - `style.sass`: Главный файл, импортирующий все остальные части.
  - `_general.sass`, `_fonts.sass`, `_media.sass`: Разделы стилей.
- `it_gid/js/`: Папка для JavaScript-файлов.
  - `scriprt.js`: Основной скрипт с кастомной логикой.
- `it_gid/assets/`: Ресурсы проекта (шрифты, изображения).

## Запуск и настройка

1.  **Клонируйте репозиторий** на свой локальный компьютер.
2.  **Компиляция SASS**: Для отображения стилей необходимо скомпилировать `it_gid/sass/style.sass` в `it_gid/css/style-min.css`. Это можно сделать с помощью любого SASS-компилятора или плагина для редактора кода, например, **Live Sass Compiler** для VS Code.
    *   *Важно:* Настройки компилятора должны быть установлены так, чтобы итоговый файл сохранялся в `it_gid/css/style-min.css`.
3.  **Просмотр**: Откройте файл `it_gid/index.html` в любом веб-браузере. Для удобства разработки можно использовать плагин **Live Server** для VS Code.
4.  **Просмотр на GitHub Pages**:
    Проект размещен на GitHub Pages, вы можете просмотреть его по адресу: `https://pvllsv.github.io/it_gid/`
